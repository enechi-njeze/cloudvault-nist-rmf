## Purpose

This folder contains all visual diagrams that support the NIST RMF implementation documentation for CloudVault FHX. Diagrams are a required component of any professional authorization package because they communicate authorization boundary scope, process flow, and control relationships in ways that lengthy prose cannot. Senior reviewers including AOs and JAB board members review diagrams first, before reading SSP sections.

## Diagram Standards

All diagrams in this folder must meet the following standards:

- Legible at standard monitor resolution (1920x1080)
- Exported as PNG at minimum 150 DPI
- Include a title block with system name, diagram title, version, and date
- Use a consistent color scheme: navy blue for primary elements, teal for secondary, gold for callouts
- Include a legend where color coding or shapes carry meaning
- Use the filename convention shown in the table above exactly
